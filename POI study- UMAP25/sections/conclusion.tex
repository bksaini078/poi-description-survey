\section{Conclusion}


This study provides evidence that large language models can personalize point-of-interest descriptions in ways that improve how users judge travel information. Compared to manual baselines, AI-adapted descriptions were rated as clearer and showed borderline evidence of improved trustworthiness, while perceived significance remained comparable. Optional post-study feedback suggests that participants were generally comfortable reading AI-generated descriptions in this setting.

These findings suggest that personalization can support travel decision-making by foregrounding details that match user interests while preserving core facts about a place. For travel platforms, LLM-based rewriting offers a practical path to scaling tailored micro-content without extensive manual editing. More broadly, results highlight that description framing can shape perceived information quality alongside which places are recommended.
